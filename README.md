# 🚀 Visualizer Performance GitHub

Um dashboard simples e moderno para visualizar a quantidade de commits e Pull Requests de um usuário do GitHub utilizando a API REST do GitHub.

## 🛠️ Tecnologias Utilizadas
- **HTML5**
- **JavaScript (ES6+)**
- **Tailwind CSS** (Estilização)
- **GitHub REST API**

## 📦 Como usar
1. Clone este repositório.
2. Abra o arquivo `index.html` no seu navegador.
3. Insira seu nome de usuário e seu **Personal Access Token (PAT)** do GitHub.
4. Clique em "Analisar Perfil".

## ⚠️ Segurança
Este projeto roda inteiramente no lado do cliente (Frontend). Seu token é enviado diretamente para a API do GitHub e não é armazenado em nenhum servidor externo.
